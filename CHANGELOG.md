# Changelog

All notable changes to this project will be documented in this file.

## [1.0.0] - 2026-04-24

Initial stable release of ShinBot.

- Established the core Agent and Bot orchestration architecture around Prompt Registry, Tool execution, commands, sessions, and plugin lifecycle.
- Shipped the FastAPI-based management API and the Vue 3 dashboard for runtime administration and observability.
- Added built-in adapters, prompt-definition management, media tooling, persistence, and a broad automated test suite to support the first stable release.
