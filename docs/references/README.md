# External References

`docs/references/` 存放外部项目、协议或规范资料的本地镜像。这里的内容只作为参考材料，不代表 ShinBot 当前架构或实现承诺。

## 当前资料

- `satori-docs/`
  - Satori 协议与资源模型文档镜像。
  - ShinBot 的统一事件、资源和消息 AST 设计会参考 Satori，但实际支持范围以 `../design/`、`../internals/` 和当前源码为准。

## 使用规则

- 引用外部协议事实时，可以链接到这里。
- ShinBot 自己的设计结论应写入 `../architecture/` 或 `../design/`，不要只停留在外部参考目录。
- 不在这里混入项目阶段总结、实现计划或本地环境记录。
