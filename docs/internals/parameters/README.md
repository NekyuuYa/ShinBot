# Internal Parameters

这个目录用于记录当前实现里仍然重要、且会明显影响行为或性能的内部参数。

这些参数的共同特点：

- 它们已经在代码里发挥实际作用。
- 它们会影响打包、缓存、压缩、淘汰或运行时行为。
- 它们暂时不一定适合直接暴露给普通用户修改。
- 它们的默认值和语义可能会随着实现优化而调整。

维护约定：

- 这里只记录“当前代码中的真实行为”，不记录尚未实现的理想设计。
- 如果某个参数后来被做成实例配置或前端设置，需要在文档里同步标记“已暴露”。
- 如果某个参数已经不再重要，或者只是纯局部实现细节，就不应继续保留在这里。
