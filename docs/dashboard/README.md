# Dashboard Docs

`docs/dashboard/` 记录 Dashboard / WebUI 的实现约定、设计审查结果和前端补充说明。

Dashboard 的长期接口规范仍放在 `../design/interfaces/`：

- `../design/interfaces/webui_design_spec.md`
- `../design/interfaces/api_communication_spec.md`
- `../design/interfaces/model_runtime_webui_spec.md`

## 当前文档

- `design-review.md`
  - Dashboard 现有界面的设计审查记录和待处理问题。
- `i18n/structure.md`
  - Dashboard 国际化目录结构、键名分层和维护约定。

## 放置规则

- 页面交互、前端状态管理和 i18n 维护说明可以放在这里。
- 前后端 API 契约、模型运行时 WebUI 规格等长期规范放在 `../design/interfaces/`。
- 一次性截图审查、临时验收记录和施工笔记不进入主文档树。
